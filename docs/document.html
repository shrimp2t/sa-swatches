Admust - A WordPress admin template for WooCommerce

Admust gives you an e-commerce focused panel by simplifying and improving WordPress Admin and WooCommerce using Rest API and React.

Features

  • White label
  • Clean up your admin and focuses on e-commerce.
  • New interface to manage orders, products, customers, coupons, comments and reports.
  • Quick view/edit products
  • Quick view/edit orders with purchased items thumbnails
  • Global search for orders, products and customers, menu from anywhere
  • Notification center.
  • Advanced menu navigation.
  • Remove all annoying notifications
  • End more...

Plugin Installation

Before starting, please install and activate WooCommerce plugin

  1. Login to your WordPress
  2. Go to Plugins -> Add New
  3. Click Upload Link
  4. Browse to the aadmust-v.xxxx.zip file and choose that file
  5. Click "Install Now" button
  6. Wait while plugin is uploading to your server
  7. Click "Activate Plugin" button
  8. Click to button "View new admin interface" to use new admin

After plugin activated you can access to new admin interface by url: https://you_site.com/myadmin?admust_fist_load=1

FAQs

Where can i find WordPress or Plugins/Themes menu?

You can find them in the Apps menu or use search form to search menu that you want to find.

How can i access to WordPress interface?

Just add param no_admust to your admin url
Example for plugins page: https://your_site.com/wp-admin/plugins.php?no_admust
