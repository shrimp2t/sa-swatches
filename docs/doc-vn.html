SaSwatches plugin features

  • Auto Convert All product attribute dropdowns to button swatch by default
  • Enable color swatches for variable product attribute options
  • Enable image swatches for variable product attribute options
  • Enable label/text swatches for variable product attribute options
  • Convert existing product attribute options into color, image, and label swatches
  • Display swatches in catalog / archive / product page
  • Multiple color for product attribute option
  • Variation swatches size controls
  • Customizable variation option text/label
  • Customizable variation text/label
  • Swatches tooltip
  • Swatches in drawer
  • Show/hide swatch label
  • Different swatch settings for each attribute type
  • Different swatch settings archive and single product page
  • Overwrite product attribute settings from product page.
  • Overwrite product attribute option settings from product page.
  • Drag and sort attribute option in product settings page
  • Support all product custom attributes

Plugin Installation

Before starting, please install and activate WooCommerce plugin

  1. Login to your WordPress
  2. Go to Plugins -> Add New
  3. Click Upload Link
  4. Browse to the sa-swatches-v.xxx.zip file and choose that file
  5. Click "Install Now" button
  6. Wait while plugin is uploading to your server
  7. Click "Activate Plugin" button
  8. Click to button "View new admin interface" to use new admin

After plugin activated you can access to new admin interface by url: https://you_site.com/myadmin?admust_fist_load=1

FAQs

Where can i find WordPress or Plugins/Themes menu?

You can find them in the Apps menu or use search form to search menu that you want to find.

How can i access to WordPress interface?

Just add param no_admust to your admin url
Example for plugins page: https://your_site.com/wp-admin/plugins.php?no_admust
