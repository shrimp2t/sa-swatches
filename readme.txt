== Description ==


Add more blocks with advanced settings for Gutenberg - Blocks Editor.

### Features
Live preview for blocks.
**Advanced section and column (Row) Block with more settings:**
- Number columns. 
- Column gap with responsive settings.
- Min height with responsive settings.
- Drag to resize column width.
- Custom column with responsive settings. 
- Vertical Align with responsive settings.
- Horizontal Align  with responsive settings.
- Dimensions with responsive settings (margin, padding).
- Advanced styling settings
	- Text color.
	- Link color settings (normal and active).
	- Border with responsive settings
	- Advanced background settings.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/blocksify` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
